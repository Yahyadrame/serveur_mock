# User Service - Mock Server

Serveur mock généré à partir du contrat OpenAPI `openapi.yaml` (Alumni Platform - User Service).

## Lancer en local
npm install
npm start
# -> http://localhost:8080/docs (Swagger UI)

## Déploiement (Render.com, gratuit)
1. Pousser ce dossier dans un repo GitHub.
2. Sur render.com -> New -> Web Service -> connecter le repo.
3. Build command: npm install
   Start command: npm start
4. Render fournit une URL du type https://<nom>.onrender.com
